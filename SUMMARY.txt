🚀 CARA MENJALANKAN:

1. VERSI OPENCV (RECOMMENDED):
   python face_detection_opencv.py

2. VERSI MEDIAPIPE (ADVANCED):
   python main.py

3. INSTALLER:
   python install.py

🎯 FITUR:
- Deteksi wajah real-time
- Klasifikasi ekspresi: Senang, Sedih, Marah, Neutral
- Mode webcam dan gambar
- Visualisasi bounding box dan landmarks

⚙️ DEPENDENCIES TERINSTALL:
- OpenCV 4.5.5 ✓
- NumPy 1.21.6 ✓
- MediaPipe (optional) ⚠️

💡 TIPS:
- Mulai dengan face_detection_opencv.py (lebih stabil)
- Gunakan pencahayaan yang cukup
- Jarak optimal: 50-100cm dari kamera
- Tekan 'q' untuk keluar dari mode webcam

🆘 TROUBLESHOOTING:
- Jika webcam error: tutup aplikasi lain yang menggunakan kamera
- Jika import error: jalankan install.py
- Jika performance lambat: gunakan versi OpenCV
